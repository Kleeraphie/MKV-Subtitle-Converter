Help for choosing the right settings

Format of the new subtitles:

The format of your new subtitles.

Edit subtitles before muxing:

Before exchanging the subtitles with the new ones, the program will pause so you can edit them.

Save images of PGS subtitles:

You can save the images that make up the old subtitles. This can be useful for debugging.

Keep original MKV files:

You can keep the original MKV files after conversion.

Keep a copy of the new subtitle files:

You can save a copy of the new subtitle files separately.

Use different languages for some subtitles:

This can be useful if the English subtitles contain letters like ä, ö or ü. Then you could use the German language because it contains all characters.

Allowed text color brightness deviation:

You can change the maximum allowed difference in brightness in the text color. This can be useful because some subtitles are not perfectly white.
